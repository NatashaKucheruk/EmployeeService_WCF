---------------------------------------------------------------------------------------------------------------
To automatically create the database, use one of the batch files located in the Batch(Version) folder, 
inserting your database connection string.
Alternatively, you can create the database manually by running the SQL scripts from the sql folder.
----------------------------------------------------------------------------------------------------------------
I used manual dependency injection to keep the project simple and dependency-free, as this is a small test project.
In a larger or production-grade system, I would use a DI container like Ninject 
or Microsoft.Extensions.DependencyInjection for better scalability and maintainability.
